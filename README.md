## How to build

1. Clone this git repository and change to the project directory

```bash
git clone https://github.com/t4top/mina-wordle-game.git
cd mina-wordle-game
```

2. Install project dependencies

```bash
npm install
```

3. Follow the build steps under each part

- [oracle](oracle/)
- [contracts](contracts/)
- [ui](ui/)
